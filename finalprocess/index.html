Final Project Process Visualizations

There are 3 main changes I want to implement to my Studio 2 project. First, I plan to continue editing the image map to hover and "highlight" the items selected using layered images. This will create the effect that an item is being selected more dynamically. Secondly, I will improve the user experience of the modal as it pops up by adding a fade in and zoom in transition upon opening up the overlay. Lastly, to continue the horror feel of the website, I plan to add a "spotlight" effect where the mouse is at, the screen is brighter at that area, much like a flashlight viewing lost loot of a horror game!
